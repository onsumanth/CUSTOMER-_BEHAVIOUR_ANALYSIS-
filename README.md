# Customer Behaviour Analysis

A data analytics project showcasing customer shopping behavior analysis using **Python, SQL, and Power BI**.

This project demonstrates an **end-to-end data analytics workflow** — from raw data cleaning and exploration to SQL analysis and interactive dashboard visualization.

---

## Project Overview

Retail businesses generate large amounts of transactional data. Understanding this data helps companies identify purchasing trends, improve marketing strategies, and increase customer retention.

The goal of this project is to analyze customer shopping data and answer key business questions such as:

* Which customers generate the most revenue?
* Do discounts influence spending?
* What products receive the best reviews?
* Do subscribers spend more than non-subscribers?
* Which customer segments drive the most revenue?

The project simulates a **real-world business analytics scenario** where data is transformed into actionable insights.

## Project Workflow

### 1. Data Loading (Python)

The dataset was imported using **Pandas** in a Jupyter Notebook.

```python
import pandas as pd
df = pd.read_csv("customer_shopping_behavior.csv")
```

---

### 2. Exploratory Data Analysis (EDA)

Performed initial analysis to understand:

* Data structure
* Data types
* Missing values
* Distribution of features

Key steps included:

* `df.info()`
* `df.describe()`
* Checking null values

---

### 3. Data Cleaning

The following preprocessing steps were applied:

* Handled missing values in **Review Rating**
* Standardized column names
* Removed redundant columns
* Created new features such as:

**Age Groups**

* Young
* Adult
* Senior

**Customer Segments**

* New
* Returning
* Loyal

---

### 4. SQL Analysis

The cleaned dataset was loaded into a **SQL database** to perform business analysis.

Key business queries included:

* Revenue generated by **gender**
* Customers using **discounts but spending above average**
* **Top-rated products**
* **Shipping type impact** on purchase amount
* Revenue comparison between **subscribers and non-subscribers**
* **Customer segmentation** based on purchase history
* **Top 3 products per category**
* Revenue contribution by **age group**

Example SQL Query:

```sql
SELECT gender, SUM(purchase_amount) AS revenue
FROM customer
GROUP BY gender;
```

## Key Insights

Some major insights from the analysis:

* Female customers generate slightly higher revenue than male customers
* Express shipping customers spend more per transaction
* Subscription customers spend more than non-subscribers
* Loyal customers contribute a large portion of revenue

## Business Recommendations

Based on the analysis:

* Promote **subscription programs** to increase customer lifetime value
* Introduce **loyalty programs** for repeat buyers
* Highlight **top-rated products** in marketing campaigns
* Use **targeted marketing strategies** for high-value customer segments
* Optimize **discount strategies** to balance revenue and profit
